Car Number Plate Detection

Developed a website that detects and stores the Number plates of cars captured from a CCTV using Opencv and Optical Character Recognition (OCR). It is build using some python Libraries, we had used SMTP protocol for transferring of messages about the car details to the users, who have registered in our website. This developed model acts a real-world prototype and can be implemented at Traffic-signals, Apartments, Multiplexes, Malls and so...on.
